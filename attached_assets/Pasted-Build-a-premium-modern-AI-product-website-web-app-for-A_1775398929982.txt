Build a premium, modern AI product website + web app for “Ath.ai” using Next.js, React, Tailwind CSS, Framer Motion, and a proper backend/auth/database setup.

Product:
Ath.ai is an AI platform that lets users upload a single photo of a furniture item and generate a high-quality 3D model from it. Users should be able to preview the generated 3D model, manage their generations, and download exported files from their account.

Critical product rules:
1. Users must create an account or sign in before they can generate any 3D model.
2. Users must create an account or sign in before they can download any 3D model.
3. Generated outputs must be stored in the user’s profile/account.
4. Each user should have a personal dashboard where they can view previous uploads, generated models, statuses, previews, and downloadable files.
5. The UI should encourage signup naturally before generation begins.

Main experience:
The product should feel like a real AI platform with two connected parts:
1. a premium marketing website
2. a logged-in app/dashboard experience

Core user flow:
1. Visitor lands on homepage
2. Visitor sees premium product explanation and demo
3. Visitor clicks upload / try demo / generate
4. If not authenticated, they are redirected to signup/login
5. After account creation, user enters app/dashboard
6. User uploads a furniture image
7. System simulates or handles AI processing
8. A generated 3D model appears in an interactive viewer
9. The result is saved to the user’s account
10. User can revisit prior generations from their profile/dashboard
11. User can download their generated 3D models only while signed in
